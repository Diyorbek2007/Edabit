## 46. Return a String as an Integer

Create a function that takes a string and returns it as an integer.

**Notes:**

- All numbers will be whole.
- All numbers will be positive.
 
**Examples:**

```js
    stringInt("6") ➞ 6

    stringInt("1000") ➞ 1000

    stringInt("12") ➞ 12
```
